About This Project

Decoding Nature, the base course of this project, analyzes what has happend in the past, or what is happening in present, and understand it, allowing us to better understand future events.

As we carried on with the course, and started understanding how to 'decode' and 'understand' nature, one question occurred. "Would I be able to 'decode the nature' of NYUAD?"

With this, I started this project with the plan to get inputs of students' thoughts by displaying certain keywords and displaying what sutdents think.

Status Quo & Limitation

While the initial plan was to get inputs from the student body, due to certain constrainsts, it is not possible to see the whole student body's word cloud yet.

Currently, the website allows users to create their own word cloud around the keyword 'NYUAD.' Then, when they add a new cloud, their friend will be able to create their own cloud as well. This will allow users to compare different word clouds people have.

In the future, the functionality of getting user inputs and displyaing them rather than personally set inputs will give a better view of what people might think regarding the keywords.

If interested in the project, please keep an eye on how this project progresses.
